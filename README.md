# vim
VIM configuration (.vimrc) and native VIM plugin management

## Use this VIM configuration
```
$ mkdir -p ~/git
$ cd ~/git
$ git clone https://github.com/backloop/vim.git
$ cd ~
$ ln -s ~/git/vim/.vimrc .vimrc
```

## Add plugins
```
$ vi ~/git/vim/install.sh
```

## Upgrade plugins
```
$ ~/git/vim/install.sh
```
